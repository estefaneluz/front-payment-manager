## Sobre
> Uma aplicação para cadastrar e gerenciar cobranças.

## Tecnologias 

```
- React.js
    - React Hook Form
    - React input mask
    - React router dom
- Material-ui
- Dayjs 
- Integração com a API via fetch
``` 

## Funcionalidades

```
- Login
- CRUD de usuários
- CRUD de clientes
- CRUD de cobranças 
- Ordenação por nome
- Pesquisa 
- Relatórios
- Tratamento de erro
```
